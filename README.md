# Tour Sharing Platform

A full-stack web application that allows users to discover, create, and share travel tours with the community. Developed as a graduation project, the platform provides a secure and interactive environment where users can explore destinations, upload tour information, and engage through comments and ratings.

---

## 📖 Overview

The Tour Sharing Platform is designed to connect travelers by providing a space to share travel experiences and discover new destinations. Users can create personalized tour posts, upload images, interact with other users, and manage their own content through a secure authentication system.

This project was developed as part of the Bachelor's Degree in **Information and Network Security**.

---

## ✨ Features

- User registration and login
- Secure authentication and authorization
- Role-based access control
- Create, edit, and delete tours
- Upload tour images
- Store images using Cloudinary
- Browse tours shared by other users
- Search and explore destinations
- Comment on tours
- Rating system
- Responsive user interface

## 🚀 Getting Started

### Prerequisites

- Node.js
- npm
- MongoDB
- Cloudinary account

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/tour-sharing-platform.git
```

Install dependencies.

Frontend:

```bash
cd client
npm install
```

Backend:

```bash
cd server
npm install
```

---

## ▶️ Running the Project

Start the backend server:

```bash
cd server
npm start
```

Start the frontend:

```bash
cd client
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
